cmFileReader
directive in module _directives

Description

On input event change, call a uploadStore function (readFile) to read the file

Dependencies

  • uploadStore

Usage

as attribute
<ANY cm-file-reader
     callback-file="{Function}">
   ...
</ANY>

Directive info
