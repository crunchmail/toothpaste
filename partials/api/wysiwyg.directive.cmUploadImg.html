cmUploadImg
directive in module wysiwyg

Description

Upload a image and modify image properties

Dependencies

  • $log
  • $timeout
  • uploadStore
  • wysService
  • Lodash

Usage

as element:
<cm-upload-img
       el="{Node}"
       parent-el="{Node}">
</cm-upload-img>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
elNode

Passing element to modify

parentElNode

Passing parent element to modify

Methods

  • addAltImage()

    Add a description alt on image

  • addLinkImage()

    Add a link on image

  • changeSizeImage()

    Change the image width

  • removeAltAttr()

    Remove description alt on image

  • removeLinkAttr()

    Remove link image

  • uploadImg(file)

    Post an image to API

    Parameters
    ParamTypeDetails
    fileFile

    input file value
