cmColorModal
directive in module wysiwyg

Description

Color modal, to change text color

Dependencies

  • $rootScope
  • wysService

Usage

as element:
<cm-color-modal
       el="{Object}"
       on-success="{Function}"
       color="{Object}"
       callback-fct="{Function}">
</cm-color-modal>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
elObject

Passing element

onSuccessFunction

Promise success function

colorObject

Passing color

callbackFctFunction

The callback function

Methods

  • applyColor()

    Apply the color on element and launch callback function and dialog success promise

  • changeColor(color)

    Change text color

    Parameters
    ParamTypeDetails
    colorObject

    color object
