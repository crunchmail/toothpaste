cmWysiwygButton
directive in module wysiwyg

Description

Upload a image and modify image properties

Usage

as element:
<cm-wysiwyg-button
       el="{Node}"
       config="{Object}"
       callback="{Function}">
</cm-wysiwyg-button>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
elNode

Passing element to modify

configObject

Passing element config

callbackFunction

Callback function

Methods

  • changeBgColorButton()

    Function to change button background-color and button border-color

  • changeBgColorButton()

    Function to change button color

  • changeLink()

    Function to change link

  • changeText()

    Function to change size text

Properties

  • button

    Init scope button

  • protocols

    Init all protocols
