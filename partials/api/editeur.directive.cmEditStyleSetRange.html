cmEditStyleSetRange
directive in module editeur

Description

A directive to edit style set rule with a input range

Dependencies

  • editStyleSetFactory
  • $rootScope

Usage

as element:
<cm-edit-style-set-range
       min-range="{Number}"
       max-range="{Number}"
       step-range="{Number}"
       regexp="{String}"
       init-value="{Object}">
</cm-edit-style-set-range>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
minRangeNumber

Min range for input range

maxRangeNumber

Max range for input range

stepRangeNumber

Step range for input range

regexpString

Regexp to find style rule

initValueObject

To init with the correct value

Methods

  • resetValue()

    Reset the value, and re-apply old value

  • resetValue()

    Apply the new style
