cmFontSizeModal
directive in module wysiwyg

Description

Font size modal to change the propertuy font-size on an element

Dependencies

  • $rootScope
  • wysService

Usage

as element:
<cm-font-size-modal
       el="{Object}"
       on-success="{Function}"
       callback-fct="{Function}">
</cm-font-size-modal>

Directive info

  • This directive creates new scope.

ParamTypeDetails

callbackFctFunction
