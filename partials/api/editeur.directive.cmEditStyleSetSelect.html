cmEditStyleSetSelect
directive in module editeur

Description

A directive to edit style set rule with a select

Dependencies

  • editStyleSetFactory
  • $rootScope

Usage

as element:
<cm-edit-style-set-select
       regexp="{String}"
       select-value="{Object}"
       init-value="{Object}">
</cm-edit-style-set-select>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
regexpString

Regexp to find style rule

selectValueObject

Init select

initValueObject

Init select value

Methods

  • change()

    Apply the new style

  • resetValue()

    Reset the value, and re-apply old value
